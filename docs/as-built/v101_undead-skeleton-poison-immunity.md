# v101 As-Built — Undead Skeleton Poison Immunity

Date: 2026-06-12
Codename: `undead-skeleton-poison-immunity`

## What Shipped

- Added `dungeon_undead`, localized as Skeleton / Esqueleto, with `resistances.poison = 1.0`.
- Added the `undead_skeleton_poison_immunity` lab world and protocol bot scenario.
- Added `monster_skeleton_v0`, generated by the deterministic GLB pipeline, with required biped
  skeleton nodes and a pinned manifest `sha256`.
- Added `monster_skeleton` to the shared monster visual catalog, schema, Godot scene registry, and
  headless animation scene coverage.
- Changed Poison Stab DOT application to start from a connected poison-applying hit even when final
  mitigated damage is zero.
- Allows poison DOT setup with zero source damage so poison-immune targets can remain poisoned and
  emit authoritative `monster_damaged` events with `damage_type: "poison"` and `damage: 0`.
- Expanded bot event matching so `event_count` assertions can filter by `damage_type`, `damage`,
  `raw_damage`, and `mitigated_damage`.
