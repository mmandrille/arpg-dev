1. **Shared rules are the single point of failure and the single point of repair.** The Rage/Volley/Split Arrow drift propagated consistently because Go, protocol bot, and client all consume the same catalog.
2. **CI is doing its job.** The gate caught semantic regressions and ratchet drift that simpler schema validation could not.
3. **The v190 Sanctuary presentation issue is resolved.** `32c9d347` moved Sanctuary radius ownership back to `SkillRulesLoader` and added focused client coverage.
