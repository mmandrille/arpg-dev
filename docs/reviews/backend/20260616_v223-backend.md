## 2. Technical

2. Convert `validateMonsterRarities` expected weights/colors/scales into a named golden or schema-backed catalog validator.
3. Split the next touched skill/class assertion cluster out of `server/internal/game/game_test.go`.
4. Continue extracting store domains out of `server/internal/store/repos.go` when new persistence features touch it.
